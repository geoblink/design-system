`GeoPill` is a component which just renders some inline content inside a round
bordered pill.

```jsx
<div class="element-demo">
  <h3 class="element-demo__header">Default variant</h3>
  <div class="element-demo__block" style="justify-content: space-around;">
    <geo-pill>Pill's content</geo-pill>
  </div>
  <h3 class="element-demo__header">Light variant</h3>
  <div class="element-demo__block" style="justify-content: space-around;">
    <geo-pill variant="light">Pill's content</geo-pill>
  </div>
  <h3 class="element-demo__header">Dark variant</h3>
  <div class="element-demo__block" style="justify-content: space-around;">
    <geo-pill variant="dark">Pill's content</geo-pill>
  </div>
  <h3 class="element-demo__header">Opaque variant</h3>
  <div class="element-demo__block" style="justify-content: space-around;">
    <geo-pill variant="opaque">Pill's content</geo-pill>
  </div>
</div>
```
