`GeoBorderedToken` is a component which just renders a small static label in a button like style.

```jsx
<div class="element-demo">
  <div class="element-demo__block" style="justify-content: space-around;">
    <geo-bordered-token
      :icon="['fas','map-marker']"
      label="Geoblink"
    />
  </div>
</div>
```
